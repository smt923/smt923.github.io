About me

My name is Jake, otherwise known as 'smt, and I'm a programmer (and a bunch of other stuff) from England. I've been using some sort of computer for as long as I remember and programming for the last 6 or so years.

I have a wide range of interests and hobbies when I'm not programming, these include things from gaming, esports and infosec to music, playing the guitar, shortwave radio and photography.

My work & stuff I've done:

Github - Published many small-medium open source projects, contributed documentation and small changes to projects like Rust and Unreal Engine 4 as well as many libraries across many languages.

Ludum Dare - Helped plan, design and develop small games for the Ludum Dare game jam, using Unity and C# and working to an incredibly tight deadline in a small remote team, our team dynamic has been pretty fluid though I took up a lot of the programming work, often helped by the rest of the team.

Languages I've used the most:

C#, Go, Python, Rust, Lua, a little Javascript, Haskell and Elixir

Stuff I have experience with:

A broad range of utilities and command line applications, web scraping & task automation, Unity3D, simple emulators, interpreters & compilers, backend web development, RESTful APIs, Elasticsearch/Kibana, Docker, Cloud hosting providers, Linux server setup & management

Projects

In my spare time I enjoy taking on small to medium programming projects, often making utilities for friends or myself, or spending my time researching and learning about things such as emulation or creating interpreters for simple toy languages.

This is a short list of some of the projects from my Github, as well as small games for various game jams:

General

tinytwitch

A low resource, command line Twitch.tv chat viewer, with capabilities to log chat and display things in a concise and resource friendly manner for streamers.

fortnite-mouse-accel

A tool written very early on for Fortnite, allowing you to easily edit mouse acceleration settings before they were fully added to the game. This tool proved mildly popular, gaining a few reddit threads and small online articles written about it, continuing to drive traffic to the project even though it's basically obsolete.

chip8

CHIP-8 emulator/virtualmachine, including graphics, made in Rust to both learn more about emulation as well as to get some practice using Rust on a larger project.

gobfk

Brainfuck interpreter with an interactive mode, written in Go, written to refine some of my new knoweldge on simple interpreters and learn how to use Go for it

twitch-classifier

An early and in progress attempt at making a tool to detect the slightly more subtle and annoying comments in a twitch chat, a first attempt at using some machine learning in a project.
